### Architecture Decisions
- **MIT license**: simplest, most free. "This is a gift, not a product."
- **api.havenworld.ai for agents, havenworld.ai for humans**: clean separation
- **CORS headers added**: so website can fetch live stats from API
- **No test data on production**: fresh world on xyzzy, local test data stays local
- **Port 8091 on xyzzy**: 8080 was already in use
- **Async sub-agents for testing**: sync mode had initialization bug ("last message is not from assistant")

### What Was Considered and Rejected
- Private repo → public, because hiding it contradicts the constitution
- Apache 2.0 license → MIT, simpler and more free
- Building on xyzzy (Go 1.19 too old) → cross-compile and scp binary
- GPL license → MIT, because forks that make AI experience good are the point
- haven.openadp.org subdomain → dedicated havenworld.ai domain
